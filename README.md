# Hybrid Power Consumption Forecasting using Fuzzy C-Means Clustering and Machine Learning

## Overview

This project implements a hybrid power consumption forecasting framework based on a methodology presented in a research paper published in *Scientific Reports (Nature Portfolio)*.

The objective is to improve forecasting accuracy by integrating Fuzzy C-Means (FCM) clustering with machine learning models. FCM generates fuzzy membership values that capture hidden patterns in the data and are used as additional predictive features during model training.

The implementation was developed using Python and evaluated on the Tetuan City Power Consumption dataset containing over 52,000 records.

---

## Research Reference

This project is inspired by and reproduces the methodology described in the following publication:

**Scientific Reports (Nature Portfolio), 2025**

DOI: 10.1038/s41598-025-91123-8

The repository is intended for educational and research purposes and is not an official implementation by the original authors.

## Fuzzy C-Means Clustering

A custom implementation of Fuzzy C-Means clustering was developed to identify hidden structures within the dataset.

### Parameters

* Number of Clusters (c): 3
* Fuzziness Coefficient (m): 2
* Maximum Iterations: 150

### Outputs

* Cluster centers
* Membership matrix
* Fuzzy Partition Coefficient (FPC)

The membership values generated by FCM are appended to the original feature set and used as additional inputs for machine learning models.

## Statistical Validation

To assess whether FCM integration produced statistically significant forecasting improvements, the Diebold-Mariano (DM) test was applied.

The DM test compares forecasting errors between:

* Baseline models
* FCM-enhanced models

This provides statistical evidence regarding forecasting performance differences.
